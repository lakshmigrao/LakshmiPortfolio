My Projects

Tik Tak Toe

Tic Tac Toe is an engaging and interactive project that involves HTML, CSS, and Javascript. This project offers players two exciting game modes to choose from: the first one allows players to play against the computer, while the second one allows two players to play against each other. With its user-friendly interface and challenging gameplay, Tic Tac Toe is a great way to improve your problem-solving skills while having fun.

Book Store Project

The Book Store is a comprehensive web application that is built using various front-end technologies such as HTML, CSS, JavaScript, and REACT. It offers an easy-to-use interface that enables users to search for books based on their title, author's name, or even a word from the book's description. With its intuitive search functionality, users can quickly find the book they are looking for and then add it to their MyBooks list with just a single click. Additionally, users can access detailed information about each book by clicking on it from the search results page. The Book Store provides an efficient and user-friendly way for book enthusiasts to browse and manage their favorite books.

Recipe App using MERN stack

Recipe App is a MERN-based web app with a user-friendly HomePage for recipe search. MERN stands for MongoDB, Express, React and Node.js. Users can create accounts to save favorites, which they can easily edit, remove or add to the list. The app uses an external API to fetch diverse recipe options, and some functionality is restricted to logged-in users for added security.
